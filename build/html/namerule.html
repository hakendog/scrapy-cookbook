命名規則¶

函數與變數¶

函數與變數命名規則皆使用小寫駝峰型，字首一律小寫

第二個單字開始字首一律大寫

範例：

int setUpNetworking;
getField ();

常數¶

常數表示一個固定的值，通常不會變動

使用全部大寫

範例：

int VOLUME = 100;
int SPEED = 1000;

類別¶

類別(class)使用大寫駝峰型，每個單字字首皆大寫

範例：

public class MainActivity {...}
