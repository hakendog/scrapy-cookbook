# 快捷鍵

| 快捷鍵                    | 說明                                                         |
| ------------------------- | ------------------------------------------------------------ |
| Ctrl+Q                    | 查找快速文檔，即在另外一個窗口中打開其聲明                   |
| Ctrl+B                    | 查找該變數的定義位置                                         |
| Alt+Shift+C               | 查看工程最近更改的地方                                       |
| Ctrl+shift+<br />上下箭頭 | 把選擇的代碼往上下移                                         |
| Ctrl+F                    | 尋找                                                         |
| Ctrl+R                    | 取代                                                         |
| shift + F6                | 變更所有同一個類別或方法或變數的名稱                         |
| shift                     | 出現提示時選擇程式碼後按下 shift 可以避免使用 Enter 所產生重複程式碼的錯誤 |
